# LanTalk UI 设计

## 主窗口
主窗口采用 Telegram 风格双栏布局：
- 左侧：本机信息、搜索框、广播入口、最近会话、联系人分组、设置入口。
- 左侧：提供新建群组入口，群组会出现在最近会话中。
- 右侧：会话标题、消息列表、输入栏、发送按钮、文件按钮。

## 视觉原则
- 简洁、清爽、现代。
- 默认蓝色主题。
- 自己消息靠右，对方消息靠左。
- 广播消息与私聊消息视觉区分。
- 临时群组与永久群组在会话副标题中区分，成员数量可见。
- 文件消息使用卡片样式并显示进度；群组文件在同一张卡片上聚合显示完成、拒绝和失败数量。
- 图片消息使用缩略图卡片，点击后进入大图预览和缩放；群组图片同样使用缩略图并聚合显示投递状态。
- 头像采用昵称首字和稳定色彩，降低列表扫描成本。
- 避免重动画、大面积模糊和过多阴影。

## 当前阶段
当前已实现真实 UDP 在线用户、联系人部门分组、最近会话未读/活跃排序、多会话切换、临时群组、永久群组、群组图片/文件、群组文本逐成员端到端加密、群组离线补发、TCP 私聊、广播消息、文件请求确认、文件传输进度展示、图片消息预览/缩放、表情快捷输入、头像首字色块、明暗主题、会话/用户搜索过滤、本机状态展示和网络发现设置。后续可继续打磨附件内容加密、发布体验和更成熟的群密钥轮换。
