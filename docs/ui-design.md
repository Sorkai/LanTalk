# LanTalk UI 设计

## 主窗口
主窗口采用 Telegram 风格双栏布局：
- 左侧：本机信息、搜索框、广播入口、最近会话、在线用户、设置入口。
- 右侧：会话标题、消息列表、输入栏、发送按钮、文件按钮。

## 视觉原则
- 简洁、清爽、现代。
- 默认蓝色主题。
- 自己消息靠右，对方消息靠左。
- 广播消息与私聊消息视觉区分。
- 文件消息使用卡片样式并显示进度。
- 避免重动画、大面积模糊和过多阴影。

## 当前阶段
当前已实现真实 UDP 在线用户、TCP 私聊、广播消息、文件请求确认和文件传输进度展示。UI 仍保持 MVP 级别，后续可继续打磨主题、空状态和多机演示细节。
